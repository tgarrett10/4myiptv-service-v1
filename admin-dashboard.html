Daily Workflow

Use this page as the control center to launch the next step, not to replace the underlying systems.

1. Review New Requests

Start in Orders and confirm what came in before messaging or setup begins.

  • Check name, contact details, and service request
  • Confirm device setup versus online-only support
  • Decide the next step before moving the order forward
Review Orders

2. Send the Right Message

Use the correct template for quotes, payment instructions, follow-up, support, and renewals.

  • Select the correct response based on the order stage
  • Keep outreach clear and operationally consistent
  • Update status in the sheet after contact is made
Open Message Templates

3. Check Setup Readiness

Use backend references before you move a paid order into active setup.

  • Check the inventory tab for device availability
  • Use Lists to support standardized dropdown values
  • Use the panel links for provisioning access
Open Inventory Open Lists

4. Verify the Public Flow

Make sure the customer-facing path is still live and working after any updates.

  • Check the homepage
  • Check the request form
  • Check the success page
Open Homepage Open Request Form Open Success Page
