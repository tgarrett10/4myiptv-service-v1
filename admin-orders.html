Work Queues

Work these buckets in order so every order has a clear next step and nothing sits without movement.

New Requests

Review fresh submissions and decide the first move before the order starts drifting.

  • Confirm order type and customer details
  • Confirm device path or online-only path
  • Set the first next action in Orders

Needs First Response

Handle first contact fast so the customer knows what happens next and you do not lose momentum.

  • Send quote, clarification, or support response
  • Confirm plan, device, or setup method if needed
  • Move the order into the next real stage

Awaiting Payment

Push quote-ready orders toward payment so paid work can move into setup without delay.

  • Confirm amount and payment method
  • Send payment request using the correct template
  • Update payment status as soon as funds are confirmed

In Setup

Work paid orders that now need credentials, device prep, shipping prep, or support routing.

  • Confirm fulfillment path for the order

  • Prepare the order for completion without delay
